# custom_toast_library

Step 1. Add the JitPack repository to your build file

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

Step 2. Add the dependency
  	
    dependencies {
	        implementation 'com.github.Vignesh44ARR:custom_toast_library:1.0.0'
	}
